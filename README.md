### Getting Started

---
> Express JS application using postgress Database and Rate limiter
> create `.env` file nad add 

```
PGHOST={your DB link ....rds.amazonaws.com}
PGPORT=5432
PGDATABASE=work_samples
PGUSER=readonly
PGPASSWORD={your password}
```

check [Back End](/backEnd).
```
cd backEnd

npm run dev

```
